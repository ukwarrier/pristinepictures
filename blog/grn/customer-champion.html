Objective

Build an extremely simple, customizable app (once set-up and forget kind) that useful for many less tech-savvy senior citizens, internet-illiterate or under privileged folks - an app that helps such users to request or access help in one click at the event of an emergency.

 

The solution proposes to leverage the power of zapier automation by connecting with several apps such as messengers, GPS, location trackers, address book etc to alert contacts and integrating multiple data sources to provide a seamless flow of useful insights to the user.

Note

The proposed application may not be relevant in the developed countries. However, it would serve many in the other parts of the globe who are not essentially privileged.

  1.  Getting Started & Login Screen
    1.   Are you a registered user?
      •    YES? Go ahead login into your account.
    2.   NOT Registered?
      •    Create an account using GOOGLE or FACEBOOK account
      •    App requires privileged access to your account for accessing contact list
      •    Agree to the conditions and sign up
    3.   Welcome Screen
      •    You get a welcome screen with a quick tutorial as soon as you sign up
      •    you can optionally skip the tutorial
      •    Either you skip or you complete watching on-screen information cycler, you get the app home page
  2.   App Home Page
      • Have three options  MEDICAL EMERGENCY, DISTRESS ALERT, DIRECTORY SEARCH
      •  The scope of this project is to cover the features of MEDICAL EMERGENCY
  3.  MEDICAL EMERGENCY - Home
    1.   CALL AMBULANCE
      •    Clicking this option will first send out a message indicating the users location
      •    Ambulance call center operator can take advantage of the GPS data to arrange vehicle from users nearest location
      •    One click call to 108  kicks off dialer, initiate a call
      •    App will stop the dialer if the call was not picked up in 30 seconds
      •    Reinitiate the call in 5 seconds after the call drop.
  4.   ALERT CONTACTS
    1.    Option to alert all your contacts that are tagged “emergency contacts”
    2.    The option presents a screen where the user can optionally check or checkoff the apps via the alerts are sent. (valid in situations is the 3G network is not working or something like that, user can check off whatsapp, Skype and just go with SMS and CALL)
    3.    Click “OK” should fire the message (default / custom) to all in the list.
    4.    User can optionally change the default message text at message text settings .
    5.    The default message text will have the users identity (full name, primary and alternate contact numbers, GPS data of the location, time of call.
    6.    Another useful feature would be a autoreminder / autofollowup which I haven’t implemented at this point in time.
    7.    Default text for a particular event to be sent via all messaging channels will be same, to keep it safe and simple.
  5.   DIRECTORY SERVICES
    1.    Help search all the directory services available at the location.
    2.    User can search hospitals, clinics, diagnostic centers, elder care centers, other medical facilities and other ERT services.
    3.    App to leverage the power of already existing databases etc.
    4.    An option to customize the search services based on the country would be great  however, I haven’t dug deep into that at the moment.
    5.    To illustrate how to take advantage of the location based apps I picked up a few apps that I tried as examples. (practo.com, quikwell.com, justdial.com)
  6.   SETTINGS (configuring options)
    1.    PROFILE SETTINGS
      •     User can edit their name  as to how they want to display in the alert
      •     Put up a profile photo  change, remove etc.
    2.    MESSAGE SETTINGS
      •     User can change the default message text set for the alerts
      •     optin or optout of using location services or communicating with GPS data
    3.    CONTACT SETTINGS
      •     User can choose the apps they want to use to alert contacts. The app will use only the selected channels to fire off messages to alert contacts
    4.    CONNECTED DEVICES
      •     User can choose to use devices that are already connected or pair new devices via bluetooth
      •     Options to unfair a device as well
    5.    HELP
      •     Takes to the apps online help page
